# 📊 Maths Final Project

## 📌 Project Overview

This project is a mini research report based on the dataset **StudentsPerformance.csv**.
The project includes both theoretical concepts and practical implementation using Python.

The main goal of this project is to understand:

* Measures of Central Tendency
* Probability Basics
* Data Distribution
* Visualization
* Linear Algebra Operations

---

# 📂 Dataset Information

The dataset contains student performance records with attributes such as:

* Gender
* Race/Ethnicity
* Parental Level of Education
* Lunch
* Test Preparation Course
* Math Score
* Reading Score
* Writing Score

Dataset File:
`StudentsPerformance.csv`
